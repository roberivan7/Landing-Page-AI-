# 🚀 Landing Page de Vendas - IA

## 📌 Sobre o Projeto

Esta é uma landing page desenvolvida com foco em **conversão de vendas** para um produto baseado em Inteligência Artificial.
O objetivo principal é apresentar o produto de forma clara, persuasiva e responsiva.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** → Estrutura da página
* **CSS3** → Estilização e layout responsivo
* **PHP** → Processamento de formulários e lógica do servidor

---

## 🎯 Funcionalidades

* Layout moderno e responsivo
* Seções estratégicas para conversão (headline, benefícios, CTA)
* Formulário de contato/captura de leads
* Integração com backend via PHP
